## How to install

**Gmail:** Settings → See all settings → Signature → Create new → paste HTML version. Set as default for new emails + replies.

**Apple Mail (Mac):** Mail → Settings → Signatures → drag the HTML signature in.

**iOS Mail:** Settings → Mail → Signature → paste plain-text version.

**Outlook:** File → Options → Mail → Signatures → New → paste HTML version.

## After-job follow-up template

Send within 24 hours of every job. Personalize the first line.

```
Subject: Thanks again, [first name]

Hi [first name],

Thanks again for choosing OCS LLC. Hope the windows are looking sharp
and giving you the view you wanted.

If we earned it, the single most helpful thing you can do for a small
local business like ours is leave a quick Google review:

  → [paste your g.page short link]

If anything wasn't perfect, just reply to this email and tell me directly.
I'd much rather fix it than have you mention it in a review.

Either way, thanks for the trust.

— Barrett
[email signature]
```

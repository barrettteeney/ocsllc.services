Step 1 — One-time setup

Paste a GitHub fine-grained personal access token. Stored only on this device. Not connected

How to make a token (60 seconds)
  1. Open github.com/settings/personal-access-tokens/new
  2. Token name: OCS LLC photo uploads
  3. Expiration: 1 year
  4. Repository access: Only select repositories → barrettteeney/ocsllc.services
  5. Permissions → Repository → Contents: Read and write
  6. Click Generate token, copy the long string starting with github_pat_
  7. Paste it below. Done.
